fastrerandomize

An R Package for Fast Rerandomization Using Accelerated Computing

fastrerandomize released!

Read the accompanying package paper here.

2024-01-17

Tools

Employs JAX backend version 0.4.38, batched processing, and GPU acceleration to improve performance.

Improvements

Allows for use in cases where the sample size or number of covariates is very large.

Open-Source

Distributed under a GPL-3 license, fastrerandomize is developed and maintained publicly on GitHub.

Balanced Assignment

Generates pools of acceptable randomizations based on a given acceptance probability.

Tests and Thresholds

Conducts exact rerandomization tests and computes optimal rerandomization acceptance thresholds.

Applications

Speed ups in applications to economic and geographical experiments.

fastrerandomize was originally introduced in Connor T. Jerzak, Rebecca Goldstein. "Degrees of Randomness in Rerandomization Procedures." (2023).
