Install fastrerandomize

To install and load fastrerandomize, follow these steps:

  1. Open your terminal in your preferred R editor.
  2. If you haven't installed devtools already, begin by doing so:
    3. install.packages("devtools")
  3. Install the package from the GitHub:
    4. devtools::install_github(repo = "cjerzak/fastrerandomize-software/fastrerandomize")
  4. Load the package:
    5. library(fastrerandomize)
  5. When running code for the first time, create the computational environment:
    6. fastrerandomize::build_backend()
  6. Rerandomize!
